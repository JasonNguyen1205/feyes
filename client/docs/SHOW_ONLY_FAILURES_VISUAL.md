## Accessibility Features

### Keyboard Navigation
- Button is focusable with Tab key
- Press Enter or Space to toggle
- Focus outline visible

### Screen Reader
- Button announces: "Show Only Failures, 2 failures, button"
- State change announces: "Filtering active, showing 2 failed ROIs"
- Clear semantic meaning

### Color Independence
- Not relying on color alone (icon changes too)
- Text clearly describes state
- Active state has background change + icon change
